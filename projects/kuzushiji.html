Japanese Kuzushiji Character Translation

Duration: Personal Project Role: Developer Type: Deep Learning / Computer Vision
PyTorch CNN Deep Learning Computer Vision Character Recognition

Overview

An passion project utilizing my interest in linguistics to learn more about deep learning and computer vision. Developed a Convolutional Neural Network using PyTorch to translate historical Japanese Kuzushiji characters to their modern equivalents. Kuzushiji (崩し字) refers to older, cursive-style Japanese characters that can be difficult for modern readers to interpret. Although the characters are not widely used by younger generations, they are used by elders, often from rural areas.

Key Features

  • Convolutional Neural Network for character recognition
  • PyTorch implementation
  • Translation from historical Kuzushiji to modern Japanese characters
  • Image preprocessing and augmentation for robust training
  • Character classification and mapping system
  • Model evaluation and accuracy metrics

Technical Approach

The character translation system was built using modern deep learning techniques:

  • Data Collection: Utilized the Kuzushiji-49 dataset, a collection of 217,912 grayscale images of size 28x28 pixels, to train the model.
  • Image Preprocessing: Implemented preprocessing pipeline including normalization, resizing, and augmentation
  • CNN Architecture: Implemented a convolutional neural network architecture
  • Model Training: Trained model using PyTorch with loss functions and optimization techniques for multi-class character classification
  • Evaluation: Tested model on test set of 38,547 images to evaluate accuracy, achieving 89% accuracy.

Challenges & Solutions

Challenge 1: Character Variability
Kuzushiji characters have significant stylistic variation. I implemented data augmentation techniques and designed a CNN architecture that can handle character variations while maintaining accuracy.

Challenge 2: Limited Training Data
Historical character datasets can be limited. I used the Kuzushiji-49 dataset, which is a collection of 217,912 images of size 28x28 pixels, but this set is not balanced. I used data augmentation techniques to balance the dataset and improve the model's generalization.

Challenge 3: Similar Character Distinction
Some Kuzushiji characters are visually similar but map to different modern characters. I fine-tuned the model architecture and training process to improve discrimination between similar characters.

Results & Future Work

The Kuzushiji translation system achieved 89% accuracy on the test set.

Future work includes implementing a more complex model architecture and training process to improve accuracy. Additionally, I would like to implement a program that utilizes the model for real-time character recognition.
