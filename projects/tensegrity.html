3-Bar Tensegrity Robot

Duration: Jan. 2023 - May 2023 Role: Assistant Researcher Lab: Hybrid Dynamics Robotics Lab (HDRL)
Tensegrity PID Control IMU Research Rolling Locomotion

Overview

As an Assistant Researcher in the Hybrid Dynamics Robotics Lab at the University of Michigan, I designed a 3-bar tensegrity robot capable of controlled rolling locomotion. The robot utilized a staged movement gait system with IMU sensors and motor encoders for state estimation, controlled by PID controllers to actuate smooth transitions between different contracted shapes. This project acted as my first research project, learning the basics of both robotics and research.

Key Features

  • 3-bar tensegrity structure with controlled rolling locomotion
  • Staged movement gait system for smooth shape transitions
  • IMU-based state estimation for real-time pose tracking
  • Motor encoder feedback for precise actuator control
  • State machine with PID control for smooth shape transitions

Technical Approach

The tensegrity robot development involved several key technical components:

  • Mechanical Design: Designed 3-bar tensegrity structure with optimal cable tensioning and joint configurations for rolling locomotion
  • Sensor Integration: Integrated IMU sensors for orientation tracking and motor encoders for actuator position feedback
  • Control System: Developed PID controllers to manage smooth transitions between different contracted shapes during rolling motion
  • Gait Development: Created staged movement patterns that exploit tensegrity structure dynamics for controlled rolling locomotion
  • State Estimation: Implemented sensor fusion to combine IMU and encoder data for accurate robot pose tracking

Challenges & Solutions

Challenge 1: Tensegrity Dynamics
Tensegrity structures have complex, nonlinear dynamics that are difficult to model and control. We implemented a state machine with PID control to overcome the need for complex dynamic models.

Challenge 2: Shape Transition Control
Smooth transitions between different contracted shapes are esstential for efefctive rolling locomotion. I did thorough PID controller tuning to achieve smooth shape transitions and maintain forward motion.

Challenge 3: Sensor Integration
Combining IMU and encoder data for accurate state estimation in a rolling robot was challenging. We developed custom sensor fusion algorithms that account for the robot's unique motion characteristics.

Results & Future Work

The tensegrity robot successfully demonstrated controlled rolling locomotion in a 3-bar tensegrity structure. Future work includes implementing larger tensegrity with onboard electronics to prevent intereference. Additionally, dynamic modeling and control of the tensegrity structure is necessary for more complex locomotion.
